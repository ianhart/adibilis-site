# adibilis-site

## Git Sync Protocol

This repo is worked on from multiple Claude Code sessions simultaneously. **Always `git pull origin main` before making any changes.** After pushing, remind the user to pull in their other session.

## Project Context

Adibilis is a web accessibility platform with two products:
- **Tirani** — Automated scanning and source-code remediation engine (the compliance product)
- **AIDA** — Navigation assistant for disabled users (NOT a compliance tool)

These are legally and functionally distinct. Never conflate them in copy or code.

## Tech Stack

Vite + React + TypeScript + Tailwind CSS v4. Deployed to Vercel at https://adibilis-site.vercel.app.
